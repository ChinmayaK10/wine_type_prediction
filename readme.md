🍷 Wine Type Prediction System

This project implements an end-to-end machine learning pipeline to predict the type of wine (red or white) based on physicochemical features. The application covers the complete workflow, including data preprocessing, model training, prediction, and deployment through a web interface.

🚀 Features
End-to-end ML pipeline for classification
Input-based prediction using multiple wine attributes
Reverse mapping of encoded outputs to human-readable labels
Web interface built with Flask
Containerized using Docker for portability
Ready for cloud deployment


🧠 Tech Stack
Python
Flask
Scikit-learn
Pandas, NumPy
HTML, CSS
Docker


⚙️ How it Works
User inputs wine features through the web form
Data is processed and converted into a structured format
Trained model generates prediction
Output is mapped back to category (Red / White wine)
Result is displayed on the UI
